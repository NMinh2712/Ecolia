# SoulGem AI - Vòng Tay Năng Lượng Cá Nhân Hóa

Website tư vấn vòng tay đá năng lượng được tạo bằng AI, kết hợp cung hoàng đạo, năng lượng cảm xúc và phong cách cá nhân.

## Tính Năng Chính

- **Form 10 Câu Hỏi**: Thu thập dữ liệu toàn diện về người dùng
- **Danh Sách Đá Chặt Chẽ**: Chỉ sử dụng 11 loại đá được cho phép
- **AI Recommendation**: Sử dụng OpenAI GPT-4o để tạo khuyến nghị cá nhân hóa
- **Reading Card**: Hiển thị thẻ năng lượng với câu khẳng định
- **Design Chi Tiết**: Giao diện đẹp mắt, hiện đại, phù hợp Gen Z

## Yêu Cầu Hệ Thống

- Node.js 18+
- npm hoặc yarn
- OpenAI API Key

## Cài Đặt

```bash
# Clone repo
git clone <repo-url>
cd soulgem-ai

# Cài đặt dependencies
npm install

# Setup environment variables
cp .env.example .env.local
# Nhập OpenAI API Key vào .env.local
```

## Environment Variables

```
OPENAI_API_KEY=sk-...
```

## Chạy Ứng Dụng

```bash
# Development
npm run dev

# Production
npm run build
npm start
```

Truy cập http://localhost:3000 trên trình duyệt.

## Lưu Ý Kỹ Thuật

- Sử dụng sessionStorage để lưu dữ liệu tạm thời
- JSON response từ OpenAI được parse an toàn
- Có xử lý fallback nếu OpenAI API bị lỗi
- Tất cả nội dung hiển thị bằng tiếng Việt chuẩn dấu
